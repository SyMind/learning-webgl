## 斯坦福兔子

斯坦福兔子是一种计算机图形学领域广泛采用的3D测试模型，由 Greg Turk（英语：Greg Turk）和 Marc Levoy（英语：Marc Levoy）1994年于斯坦福大学制作。该模型包含了69451个三角形的数据，这些三角形由三维扫描仪确定，构成了一座陶瓷兔子塑像。
